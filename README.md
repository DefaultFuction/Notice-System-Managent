## 📋 Overview

Notice System Management is a clean, lightweight Notice Management System built with Spring Boot and MySQL. Designed for small businesses, schools, and organizations, it provides essential tools to manage internal announcements and notices efficiently.

---

## ✨ Features

| Module | Capabilities |
|--------|--------------|
| 📋 **Notice Management** | Publish, edit, delete, and view notices |
| 🔍 **Search & Filter** | Quick search by title or author |
| 📊 **Statistics** | View notice counts and categories |
| 🏷️ **Category Management** | Organize notices by category |
| 👤 **Author Tracking** | Track notice authors and contributions |
| 📱 **Responsive UI** | Clean, modern interface with mobile support |
| 📝 **Rich Content** | Support for formatted text content |
| ⏰ **Timeline View** | Chronological display of all notices |
| 🔄 **Bulk Operations** | Batch delete and status updates |
| 📈 **Analytics** | View count tracking for each notice |
